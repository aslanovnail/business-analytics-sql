# Business Analytics with SQL

## Overview
This project analyzes ride-share business data using SQL to generate operational and market insights.

## Objectives
- Identify top neighborhoods by ride activity
- Compare taxi companies by trip volume
- Analyze ride patterns and external factors

## SQL Skills Demonstrated
- Filtering
- Aggregation
- Joining tables
- Sorting and ranking
- Business insight generation

## Tools Used
- SQL
- PostgreSQL
- Python (optional)

## Key Insights
- Top neighborhoods generated the highest ride activity
- Some taxi operators had significantly higher trip counts than others
- Ride demand varied based on external conditions
